# AssetFlow

## Branch

## Responsibility

This branch contains the backend implementation of AssetFlow.

The backend is responsible for:

- REST API Development
- Database Design
- Authentication
- Authorization
- Business Logic
- Asset Lifecycle Management
- Booking Validation
- Maintenance Workflow
- Audit Workflow
- Notification Services
- Activity Logging

The frontend is being developed separately by another collaborator.

After both implementations are complete, the branches will be merged into the final production version.
